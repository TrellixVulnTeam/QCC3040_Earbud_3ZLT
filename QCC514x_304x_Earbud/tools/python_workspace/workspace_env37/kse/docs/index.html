KSE¶

KSE (Kymera Simulation Environment) is a Python-based command shell for interactive prototyping, testing, and debugging of Kymera operators running in a simulated environment. When the shell is started, the session is loaded with objects available for the user to interact with Kymera.

The audio subsystem is simulated using kalsim/kalcmd2 (kalimba simulator).

Chip platforms supported

  • Crescendo

  • Stretto

  • Strettoplus

  • Maorgen1

  • Maor

Environment characteristics

  • Python 3.6+

  • Windows and Linux support

Known limitations

  • Only single kalimba core is supported

  • No 32 bit Linux support

  • Hardware interfaces are simulated
