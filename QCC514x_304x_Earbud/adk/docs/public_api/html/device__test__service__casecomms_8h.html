void DeviceTestServiceCasecomms_Init ( void  )

Initialise DTS case comms client.

Registers DTS with cc_protocol to use case comms.

Any time after this call the registered DTS case comms client callbacks may be called for incoming case comms messages on the DTS channel or to report status of messages transmitted on the DTS channel by this client.
