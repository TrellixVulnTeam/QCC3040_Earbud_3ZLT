reboot_action_t SystemReboot_GetAction ( void  )

Returns the action to be taken post reboot. When app uses this function to move to a well defined state by application, the reboot action should be reset for the next boot to be as reboot_action_default_state. To achieve this the app can call SystemReboot_ResetAction.

Reboot device.
