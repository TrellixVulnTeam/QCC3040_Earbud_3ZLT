◆ ad_type_additional

ble_ad_type ad_type_additional

Additional entry in the advert to filter on if non-zero. This is useful for UUID filtering as adverts have different entry types for UUIDs. One for a complete UUID list and one for an incomplete list.

◆ find_tpaddr

tp_bdaddr* find_tpaddr

Optionally only report adverts that match the pattern if they also match the address. If resolvable addresses are used (RPA) then the public address should be used.
