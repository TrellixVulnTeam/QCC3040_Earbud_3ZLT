"is_to_host" gives the direction of endpoint (or data flow in the interface), which can be used to identify whether it is an audio streaming interface for speaker or mic. "is_to_host = 1" : to host; indicate mic interface, mic_sink is in use. "is_to_host = 0" : from host; indicate speaker interface, spkr_src is in use.
